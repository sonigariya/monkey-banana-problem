# monkey-banana-problem
Monkey Banana Problem solved using Python


🎯 Problem Statement
A monkey is inside a room:
The monkey is at the door
A box is at the window
A banana is hanging from the center of the room
👉 The banana is out of reach.


Goal:
The monkey must:
Move to the box
Push the box under the banana
Climb the box
Grab the banana


⚙️ How It Works
The program simulates:
monkey movement
Box movement
Climbing action
Final goal achievement


Output:
Monkey moves from door to window
Monkey pushes the box to center
Monkey climbs the box
Monkey grabs the banana 🍌
